# ID.me Sandbox SDK

High-integrity TypeScript SDK for ID.me Sandbox, generated by the Auth SDK Generator.

## Integration

This SDK is built on the `@cosmonexus` foundation.

### React

```tsx
import { IDmeSandboxProvider, IDmeSandboxLoginButton } from './sdk';

function App() {
  return (
    <IDmeSandboxProvider config={{ clientId: 'your-idme-client-id' }}>
      <IDmeSandboxLoginButton />
    </IDmeSandboxProvider>
  );
}
```

## Technical Details

- **Provider**: default
- **Authorize URL**: https://api.idmelabs.com/oauth/authorize
- **Token URL**: https://api.idmelabs.com/oauth/token
- **Default Scopes**: openid
- **PKCE**: Enabled (S256)
- **State Parameter**: Required
