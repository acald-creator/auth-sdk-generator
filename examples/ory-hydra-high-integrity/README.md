# Ory Hydra SDK

High-integrity TypeScript SDK for Ory Hydra, generated by the Auth SDK Generator.

## Integration

This SDK is built on the `@cosmonexus` foundation.

### React

```tsx
import { OryHydraProvider, OryHydraLoginButton } from './sdk';

function App() {
  return (
    <OryHydraProvider config={{ clientId: 'my-client' }}>
      <OryHydraLoginButton />
    </OryHydraProvider>
  );
}
```

## Technical Details

- **Provider**: default
- **Authorize URL**: http://127.0.0.1:4444/oauth2/auth
- **Token URL**: http://127.0.0.1:4444/oauth2/token
- **Default Scopes**: openid, offline_access, profile, email
- **PKCE**: Enabled (S256)
- **State Parameter**: Required
